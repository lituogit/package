### js基础

**数据类型**

在javaScript中，数据类型分为基本数据类型和引用数据类型

基本数据类型：

数字型 number         字符类型string    布尔类型 boolean   未定义类型undefined

检测基本数据类型的方法   typeof

**引用数据类型**

数组 Array  对象Object  函数 Function

**运算符 **  

算术运算符： + -* / %    =（赋值）  ++   --

逻辑运算符：&&   ||  ！  ==   ===   !=    

**循环语句**

for  (;;)     while  do while

**流程控制语句**

if  else      switch  case  break；default
